# IPartFont2::GetFontRestriction

## -description

Gets the restrictions for the font.

## -parameters

### -param pRestriction

[out]
Restrictions for the font.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
